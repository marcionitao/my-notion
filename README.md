# My Notion Application 

## A Notion private application for personal use.

### Features:
- Editing pages
- Creating text blocks
- Editing text blocks

### Technologies:
- React
- TailwindCSS
- TailwindCSS/Typography
- TipTap Editor
- React Icons
- Typescript
- Vite
